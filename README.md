# WGSL-Toy

A playground of WebGPU.

Live demo: https://valeerajs.github.io/WGSL-Toy/build/

## Features

1. support code highlight
2. support multi files droping
3. support multi code editor tabs
4. support download wgsl code
5. wgsl errors log
